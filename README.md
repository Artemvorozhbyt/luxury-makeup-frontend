# 💄 Luxury Makeup Booking — Frontend

Modern, responsive frontend for a full-stack booking platform designed for a private makeup studio.
Built with clean UI principles, multilingual support, and real-time interaction with a REST API.

🔗 **Live Website:** https://luxury-makeup-atelier.netlify.app

---

✨ Features

📅 Booking system with date & time slot selection
⚡ Real-time availability (connected to backend API)
💬 Client reviews with rating system
🌍 Multilingual interface (Ukrainian / Polish / English)
📱 Fully responsive design (mobile-first)
🎨 Luxury-style UI/UX (custom typography, layout, animations)

---

 🖥️ UI Overview

Hero section with branding and call-to-action
Services section with pricing and duration
Interactive booking form
Reviews system with dynamic rendering
Contact section with map and social links

---

🛠️ Tech Stack

HTML5
CSS3 (modular structure: variables, layout, components)
Vanilla JavaScript (ES6 modules)
Flatpickr (date picker)

---

🔗 API Integration

Frontend communicates with a custom ASP.NET Core backend:

Fetch available time slots
Create bookings
Load and submit reviews

---

🌍 Internationalization

Language switching via `localStorage`
Dynamic text replacement using custom translation system
Supported languages:

🇺🇦 Ukrainian
   🇵🇱 Polish
🇬🇧 English

---

🚀 Deployment

Hosted on **Netlify**
Automatic deploy via GitHub
Instant updates after `git push`

---

💡 Highlights

Clean separation of UI and logic
No frameworks — pure JavaScript architecture
Real production deployment
Integrated with external backend & services
